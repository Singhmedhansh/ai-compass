## Visual aesthetic

Reference: existing homepage (Hero / WizardDemo / CurationDiscipline / SunoStory / FinalCTA), Anthropic.com (editorial pacing), Linear.app (structural clarity), Stripe.com (data + editorial layering).

**Vocabulary to apply across pages:**
- Generous whitespace; sections breathe
- Editorial typography — large display headlines, restrained body
- Gradient surfaces (`bg-elev` panels, `accent-soft` washes, dimensional layering)
- Mockup chrome (the `wizard.ai-compass.in` browser frame is part of the language; can extract as reusable component)
- Calm accent usage — green only for focal positive states (CTAs, success), never decorative
- Editorial pacing — sections build narrative

## Motion design language (medium-heavy)

**In scope:**
- Page-level route transitions (Framer Motion, ~200-300ms)
- Scroll-triggered section reveals (Intersection Observer + Framer variants)
- Hover micro-interactions (card lift, button color transitions, link underlines)
- Stagger animations on lists (tool grids, collection lists, wizard results)
- Loading state choreography (skeleton → content fade)
- Selective parallax / motion-driven hero treatments on tier-1 pages

**Out of scope (defer to v2.2 if ever):**
- Custom cursor effects
- Heavy WebGL / canvas animations
- Auto-playing video backgrounds

**Motion budget:** max 3 motion treatments per page. Prevents over-engineering.

## Motion patterns observed (Phase J prework)

Notes captured during reference-site homework. Each pattern has a
mechanics sketch + AI Compass application. These feed directly
into Phase K foundation work (Framer Motion variants library,
scroll hooks).

### anthropic.com

**Pattern 1: Scroll-triggered wordmark → compact mark**
- Top of page: full "ANTHROP\C" wordmark in nav.
- After scroll past hero: collapses to "A\" compact mark.
- Mechanics: scroll position crosses a threshold (~80-120px);
  crossfade between two SVG variants via AnimatePresence;
  ~200-250ms duration; layout-stable (compact mark width matches
  wordmark slot, so no layout shift).
- AI Compass application: "AI Compass" wordmark in Navbar →
  collapses to compass-icon (or "AC" glyph) after scrolling past
  hero on every page. Polish at zero functional cost.

**Pattern 2: Hero card scale-on-reveal**
- Tier-1 hero (Project Glasswing card): image scales up dramatically
  as the section enters viewport.
- Mechanics: pinned-scroll section + `scale()` transform driven by
  scroll progress (Framer Motion's `useScroll` + `useTransform`).
  Image goes from ~0.85x to 1.0x across the pin distance.
- AI Compass application: Tier-1 page heroes — ToolDetailPage's
  hero image, DashboardPage's welcome panel, possibly Best* SEO
  templates. Adds dimensionality without being gimmicky.
